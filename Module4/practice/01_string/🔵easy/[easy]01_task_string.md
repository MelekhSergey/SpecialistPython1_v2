## "Первый и последний символ"

### Задание

Выведите первый и последний символ данной строки.

### Формат входных данных

Дана произвольная строка.

### Формат выходных данных

Вывести первый и последний символ строки.

### Решение задачи

```python
s = input("Введите произвольный текст: ")
print("Первый символ", s[0])
print("Последний символ", s[len(s) - 1])
```
